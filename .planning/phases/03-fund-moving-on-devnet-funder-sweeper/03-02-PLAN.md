<objective>
Build the sweeper — the exact-zero session->vault sweep that is structurally incapable of loading the vault secret.

Delivers SESS-06 (return remaining SOL to the vault on manual session end, exact-zero via D-05/D-06/D-07) and the sweeper half of SEC-02 (never imports vault.py; signs with the session key only; sweeps to Config.vault_pubkey). Adds the `get_token_accounts_by_owner` RPC helper (Claude's Discretion per CONTEXT) used for empty/nonzero ATA classification.

Purpose: end a devnet session (SOL + one open ATA closed first) at exactly zero lamports with every lamport (SOL + reclaimed ATA rent) landing in the vault, in a single all-or-nothing transaction.

Output: `bastion/sweeper.py`, a new `RpcClient.get_token_accounts_by_owner` method, unit tests, and the SEC-02 negative-contract assertion for the sweeper.
</objective>
